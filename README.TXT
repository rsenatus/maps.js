Maps.js est une librairie permettant de manipuler google maps. 

Une démo est disponible sur le site : http://map.getinit.fr/

FICHIERS PRINCIPAUX: 

* "maps.js" est la librairie. 
* "handle_map-unpack.js" est l'exacte copie du fichier "handle_map.js" sur le http://map.getinit.fr/ (le fichier qui sert à manipuler maps.js sur le site La seule différence c'est que  "handle_map-unpack.js" contient des annotations qui explicitent chaque ligne de code).  


FICHIERS SECONDAIRES : 

* "m-markeryellow.png" est l'image d'un "marker". 
*  "m-markerblue.png" est l'image d'un "marker" utilisé pour le roll-over. 
* "maps.min.js" est le fichier "maps.js" mais "réduit".
